Introduction

The "Flagship" Java project is inspired by the recent buzz around 12factor applications and Microservices.

The example is slightly more inspired by the notion that everyone should understand how to build Microservices from scratch with just a few dependencies. Most of the recent apps I've build tend to start with the basic concepts and app structure described below. There are also several open source projects that start with similar foundations; Spring Boot and Dropwizard are a few examples.

As applications grow (multiple apps, multiple services), I'm also a slight fan of having all the services or components within large systems reside in a single, versioned repo.

Project Structure

The example walks you through a basic Java Microservice in a few steps. For illustration purposes, I've captured the steps via the below Git tags -

git tag -ln

v0.2            Introducing Guice and App Environment
v0.3            Introducing Jetty and Jersey
